# The Alchemist Complexity Analysis

This repository contains replication materials for the article:

**“Explaining the Complexity of Jonson’s *The Alchemist*: Close Reading and Computational Analysis”**

## Contents

- `notebooks/`: Python/Colab notebooks used for the analyses.
- `data/`: input data used to generate the networks and figures.
- `figures/`: figures included in the article.
- `outputs/`: derived tables and intermediate outputs.

## Reproducibility

The analyses were run in Google Colab using Python. To reproduce the results, open the notebook in `notebooks/` and run all cells from top to bottom.
